InternBit Website Scraper

InternBit was a UH-funded research program for software developers that aimed to help ICS students in college find internships via a recommendation system. Tools used in this project were React JS, Node.js, X-Ray, Pupeteer, Tatooine, NightMare, Figma, and GitHub. We used website scrapers to collect internship opportunities from all the technology websites we could find in 2020. Our goal was to bring all tech job listings to one place, which would allow you to use filters to find your internship.
